whoami

I'm, first of everything else, a software developer.

My first blood with computers was at the age of 7 with an IBM Model 5150 donated from a family friend.

Since then i always used computers for a lot of useful and "less useful" tasks but never considered them as a future work provider before the age of 16 when i first met the C at High School.

I've been assumed at my first workplace at 7Pixel during my last bachelor's degree stage and there is where i first met the Agile manifesto, Xp Programming and the desire to always learn something new.

I currently living in Italy in Cunardo, a small town near Varese in Lombardy.

Experiences

  • 2 Years as Full stack developer on Drezzy website
  • 1 Year as backend developer and mantainer for the whole 7Pixel Network
  • 3 Years as Mobile Developer for the TrovaPrezzi Apps on iOS, Windows Phone and Android platforms
  • Currently working as Frontend developer for Kirivo

I worked since the kickstart as fullstack developer on the Drezzy platform with .NET framework based technologies like MVC framework 3.x, IIS web server and C#

As the first big project of my life i really enjoyed working on it: the biz gave us great independence and trust allowing me to learn as much as possibile on the ecommerce world.

After the Drezzy project my team was asked to move in a big technical update of the existing backend infrastructure for the whole 7Pixel network.

In that occasion i've worked with Apache Solr and Java

And now something completely different

At some point i found myself learning Android, iOS and Windows Phone for the development of native mobile apps for the TrovaPrezzi price comparison engine.

As before, my team and i received great trust from the ca$h guys and the trip went on for roughly 3 years in which i worked mainly on Android platform and server backends using Ruby.

Now i'm working in at Kirivo as a Frontend developer using Ruby.
